# Contributing to VietFuel API
*(Scroll down for Vietnamese version / Kéo xuống dưới để xem bản tiếng Việt)*

First of all, thank you for your interest in contributing to the **VietFuel API** project! This project aims to provide standard and accurate fuel prices from multiple distributors in Vietnam.

Community contributions are highly appreciated to make this project better, more robust, and easier to use.

## How Can You Contribute?
There are many ways you can contribute to the project:
1. **Reporting Bugs:** If you find a bug or experience a scraping failure, please open an Issue with detailed error logs.
2. **Suggesting Enhancements:** Have an idea for a new feature? Feel free to open an Issue to discuss it.
3. **Submitting Pull Requests (Code Contributions):**
   - Fork the repository.
   - Create a new branch for your feature or bug fix: `git checkout -b feature/your-feature-name` or `fix/your-bug-name`.
   - Ensure your code follows the existing coding style and project structure.
   - Test your code locally (especially the scraping logic to ensure no IP blocking occurs).
   - Submit a Pull Request with a clear description of the changes you've made.

## Development Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npx playwright install chromium` to install the required browser for scraping.
4. Run `npm start` to start the local development server.

Again thanks for your contribution!

# Hướng dẫn Đóng góp (Vietnamese)

Trước tiên, cảm ơn bạn đã quan tâm và muốn đóng góp cho dự án **VietFuel API**! Dự án này được xây dựng với mục tiêu cung cấp dữ liệu giá xăng dầu chuẩn xác, tổng hợp từ nhiều nhà phân phối tại Việt Nam.

Mọi đóng góp từ cộng đồng đều được trân trọng để giúp dự án ngày càng hoàn thiện, ổn định và dễ sử dụng hơn.

## Bạn có thể đóng góp bằng cách nào?
Có rất nhiều cách để bạn tham gia xây dựng dự án:
1. **Báo cáo Lỗi (Report Bugs):** Nếu bạn phát hiện lỗi logic, lỗi hiển thị hoặc phát hiện logic cào dữ liệu (scraping) bị sập do nguồn thay đổi giao diện, hãy mở một Issue kèm theo log lỗi chi tiết nhé.
2. **Đề xuất Tính năng (Enhancements):** Bạn có ý tưởng tính năng mới? Đừng ngần ngại mở một Issue để chúng ta cùng thảo luận.
3. **Gửi Pull Request (Đóng góp Code):**
   - Fork repository này về tài khoản của bạn.
   - Tạo một nhánh mới cho tính năng hoặc bản sửa lỗi: `git checkout -b feature/ten-tinh-nang` hoặc `fix/ten-loi`.
   - Viết code theo đúng cấu trúc và phong cách (coding style) hiện tại của dự án. Đừng quên giữ lại thông tin Credit bản quyền.
   - Chạy test cẩn thận ở local (đặc biệt là test bộ cào dữ liệu để tránh tình trạng Spam Request gây khoá IP).
   - Gửi Pull Request kèm theo mô tả rõ ràng về những thay đổi bạn đã thực hiện.

## Thiết lập Môi trường Phát triển (Local Dev)
1. Clone repository về máy.
2. Chạy `npm install` để cài đặt các thư viện cần thiết.
3. Chạy lệnh `npx playwright install chromium` để cài đặt trình duyệt ảo bắt buộc cho bộ cào dữ liệu.
4. Chạy `npm start` để khởi động server ở môi trường local.


Một lần nữa, xin cảm ơn sự đóng góp của bạn!
**TranQui** - *Dự án VietFuel API*
