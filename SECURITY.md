# Security Policy

## Supported Versions

Currently, only the latest major version is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** create a public issue.

Instead, please send an email to the repository owner directly. We will try our best to address the issue as quickly as possible.

When reporting an issue, please provide:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and any suggested fixes (if known).

We value the security of our users and data, and we appreciate your help in keeping this project secure.
