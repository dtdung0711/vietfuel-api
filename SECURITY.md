# Security Policy / Chính sách bảo mật
*(Scroll down for Vietnamese version / Kéo xuống dưới để xem bản tiếng Việt)*

## Supported Versions

Currently, only the latest major version is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** create a public issue.

Instead, please send an email to the repository owner directly. We will try our best to address the issue as quickly as possible.

When reporting an issue, please provide:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and any suggested fixes (if known).

We value the security of our users and data, and we appreciate your help in keeping this project secure.

# Chính sách bảo mật (Vietnamese)

## Phiên bản được hỗ trợ

Hiện tại, chỉ có phiên bản lớn (major) mới nhất được hỗ trợ các bản cập nhật bảo mật.

| Phiên bản | Được hỗ trợ        |
| --------- | ------------------ |
| 1.0.x     | :white_check_mark: |
| < 1.0     | :x:                |

## Báo cáo lỗ hổng bảo mật

Nếu bạn phát hiện ra một lỗ hổng bảo mật trong dự án này, xin vui lòng **KHÔNG** tạo một issue công khai.

Thay vào đó, vui lòng gửi email trực tiếp cho chủ sở hữu kho lưu trữ. Chúng tôi sẽ cố gắng hết sức để giải quyết vấn đề nhanh nhất có thể.

Khi báo cáo một vấn đề, vui lòng cung cấp:
- Mô tả về lỗ hổng.
- Các bước để tái hiện lại vấn đề.
- Tác động tiềm ẩn và mọi đề xuất sửa chữa (nếu biết).

Chúng tôi trân trọng tính bảo mật của dữ liệu và người dùng, đồng thời đánh giá cao sự trợ giúp của bạn trong việc giữ cho dự án này an toàn.
