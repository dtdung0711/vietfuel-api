# Chỉ mục tài liệu VietFuelAPI

## Tiếng Việt

Để dễ quản lý, toàn bộ tài liệu Markdown của dự án được gom vào thư mục `docs/`.

- Tài liệu tiếng Việt: `docs/vi/`
- Tài liệu tiếng Anh: `docs/en/`
- Điểm vào chính (VI): `README.md`
- Điểm vào chính (EN): `README.en.md`

Lưu ý:

- Template issue/PR vẫn đặt trong `.github/` vì GitHub yêu cầu đúng vị trí này để tự nhận diện.
- Không đưa log runtime, dump debug hoặc file tạm vào tài liệu chính thức trừ khi thật sự cần tham chiếu lâu dài.

---

## English

To simplify maintenance, all project markdown documents are centralized under `docs/`.

- Vietnamese docs: `docs/vi/`
- English docs: `docs/en/`
- Main entry (VI): `README.md`
- Main entry (EN): `README.en.md`

Notes:

- GitHub issue/PR templates remain under `.github/` because GitHub only auto-detects templates in that location.
- Runtime logs, debug dumps, and temporary artifacts should not be added to official documentation unless they are long-term references.
