# Code of Conduct

## Commitment

We are committed to building a friendly, safe, and respectful community environment.

## Encouraged Behavior

- Respect different opinions.
- Provide feedback in a constructive way.
- Prioritize technical facts and evidence during discussions.
- Help newcomers contribute to the project.

## Unacceptable Behavior

- Personal attacks, harassment, discrimination, or insults.
- Posting harmful content or hate-inciting material.
- Intentionally spreading misleading information.

## Violation Handling

Maintainers may:

- Ask for content changes.
- Close inappropriate issues/PRs.
- Refuse collaboration in repeated violation cases.

## Scope

This applies to issues, pull requests, discussions, and any project-related communication channels.

## Contact

To report inappropriate behavior, please create an issue with label `conduct`.
