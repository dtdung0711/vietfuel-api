# Support

If you need help, please:

1. Read `docs/en/README.md` and related documents under `docs/`.
2. Check existing issues before opening a new one.
3. Create an issue with an appropriate label:
   - `bug`
   - `question`
   - `enhancement`

## Useful Information for Support Requests

- Affected endpoint.
- Runtime environment (OS, Node version).
- Relevant error logs.
- Reproduction steps.

Thank you for supporting the VietFuelAPI community project.
